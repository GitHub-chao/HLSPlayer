# HLSPlayer
基于Flash插件的HLS播放器，支持VTT缩略图，SRT字幕，片头广告，暂停广告等


注意：播放器皮肤源文件位于工程的lib目录中


在线预览地址：http://www.fenhongxiang.com/video/vod/FPlayer.html
